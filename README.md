# Scale Trainer

A simple exercise tool for learning to construct scales.

![Screenshot](/.github/screenshot.png)

Supported scales:

- Major (+ modes)
- Minor (+ modes)
- Harmonic Minor (+ modes)
- Melodic Minor (+ modes)
- Chromatic
- Augmented
- Major Blues
- Minor Blues
- Major Pentatonic
- Minor Pentatonic

## Run

```bash
yarn && yarn start
```

Go to [http://localhost:3000](http://localhost:3000) and have keep on rockin'!
